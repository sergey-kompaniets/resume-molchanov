Pet project

Project description:

Smart Home is an abstraction over the chain of devices and the control center. Each device communicates with the center via grpc and obtains a new configuration using NATS. The center stores device’s configuration and data in the database. Device’s state can be monitored and changed via a browser. Data broadcast from the device is available through the WebSocket connection.

Project components:
  • centerms – https://github.com/kostiamol/centerms – controls all the devices and gathers metrics from them that are stored in database (Redis).
  • devicems – https://github.com/kostiamol/fridgems – is an abstraction level over appliance that produces metrics and has an ability to change some patterns of behavior — configuration.
  • dashboard – https://github.com/kostiamol/dashboard-ui
